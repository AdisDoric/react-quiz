# React Quiz
A JavaScript slide quiz built with React.

[View Demo](http://davidra.co/react-quiz/)

Quick Start
-----------

```shell
$ git clone https://github.com/davidrayoussef/react-quiz.git
$ cd react-quiz
$ npm install
$ npm start
```

App Structure
-------------

QuizApp
|__ Quiz
    |__ QuestionList
    |   |__ Question
    |       |__ Answer
    |__ Results
